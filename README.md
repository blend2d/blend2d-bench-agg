# Anti-Grain Geometry Library

Anti-Grain Geometry library, written by Maxim Shemanarev in C++. It is an Open Source, 2D vector
graphics library. Agg produces pixel images in memory from vectorial data.

# Stripped-Down

This repository contains a stripped down version of Anti-Grain Geometry that is designed to be used
for benchmarking purposes against Blend2D library. Please don't use this version for anything else.

In Blend2D project, this library is fetched into 3rdparty/agg
